# `<AGOSTINHO BORGES/>`

Portfólio pessoal de **Agostinho Borges**, desenvolvedor frontend focado
em criar experiências digitais modernas, funcionais e visualmente
elegantes.

Este projeto foi desenvolvido com **HTML, CSS e JavaScript puro**, sem
frameworks, valorizando performance, clareza de código e domínio dos
fundamentos.

------------------------------------------------------------------------

## Visão Geral

O portfólio apresenta:

-   Página inicial com introdução profissional
-   Seção de projetos em grid responsivo
-   Seção "Sobre Mim"
-   Contatos diretos (WhatsApp, Email, LinkedIn e GitHub)
-   Menu hamburger totalmente funcional
-   Cursor customizado interativo
-   Design responsivo (desktop, tablet e mobile)

------------------------------------------------------------------------

## Tecnologias Utilizadas

-   **HTML5** -- Estrutura semântica
-   **CSS3** -- Layout, animações e responsividade
-   **JavaScript (Vanilla)** -- Interatividade e comportamento do menu
-   **SVG** -- Ícones leves e escaláveis

------------------------------------------------------------------------

## Funcionalidades

-   ✅ Menu hamburger animado
-   ✅ Menu fecha ao:
    -   clicar em um link
    -   clicar fora
    -   dar scroll
-   ✅ Cursor customizado que muda de cor ao passar sobre links
-   ✅ Layout totalmente responsivo
-   ✅ Código limpo e organizado

------------------------------------------------------------------------

## Design

-   Tema escuro (dark mode)
-   Destaques em azul
-   Tipografia moderna
-   Animações sutis focadas em UX
-   Estilo minimalista e profissional

------------------------------------------------------------------------

## Como usar o projeto

1.  Clone o repositório:

    ``` bash
    git clone https://github.com/agostinho-borges24/portfolio.git
    ```

2.  Acesse a pasta do projeto:

    ``` bash
    cd portfolio
    ```

3.  Abra o arquivo `index.html` no navegador.
